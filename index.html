    The main branch is master branch, developers try not to do add code in this branch because live website code is commited in this branch. The best practice is to create another branch then merge it into this branch

  • git checkout newbranchName - call it
  • git branch --delete branchName - to delete a branch, you must be out of the branch that you want to delete.
